# Steps to run on your Local Machine
1. Clone the repository

      ```https://github.com/Abhishek-Rath/Movie_Mania.git```

2. Migrate into Movie_Mania folder

      ```cd Movie_Mania```

3. Install Dependencies

      ```npm install```

4. Start React App

      ```npm start```
